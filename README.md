# C-factor-from-Sentinel-2

This repository contains the python code to run a set of methods to determine the C-factor from time series of Sentinel-2 data. The method processes a sample of IACS field parcels by associating time series of spectral indcices to calculate the fractional vegetation cover of a field parcel through time.
